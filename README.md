# TouchBack Monorepo

## Thư mục

- backend: API Node.js (Express + MongoDB)
- frontend: React (Vite) skeleton

## Chức năng đã thiết lập

Xem README trong từng thư mục.

## Cách chạy

Mở 2 terminal:

1. Backend

```bash
cd backend
npm install
npm run dev
```

2. Frontend

```bash
cd frontend
npm install
npm run dev
```

Sau đó truy cập http://localhost:5173
